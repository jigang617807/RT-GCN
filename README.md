# Dataset Description

This folder contains the multilayer network datasets used in the experiments. Each subfolder corresponds to a two-layer network constructed from real-world data.

## Directory Structure

```text
data/
├── Bio-Chem/
├── His-Peo/
├── World-Trade/
└── fb-tt/

All datasets are organized as two-layer networks. They are used for evaluating multilayer network representation learning and link prediction methods, especially the ability of models to capture both intra-layer structural patterns and inter-layer dependencies.
